# 靜心觀呼吸

昊道文化團隊內部閱讀用的靜態學習網站。內容依《關於淨念、靜心「觀」呼吸》原文整理，網站提供：

- 首頁總覽與章節索引
- 12 個主題章節
- 原文段落閱讀
- 修煉要點、練習提醒、觀照提問
- 搜尋、觀呼吸計時器與百日練習紀錄

## 發布設定

網站以 GitHub Pages 靜態發布。

## 本機預覽

直接開啟 `index.html`，或啟動本機靜態伺服器：

```bash
python3 -m http.server 4173
```

再開啟：

```text
http://127.0.0.1:4173/
```

## 更新內容

重新抽取內容：

```bash
python3 scripts/build_content.py
```
